COMO FOI CRIADO O PROJETO "CLASH-ROYALE-TRABALHO"

Eu estava na aula de "programação mobile" no ano passado, quando o Professor "Matheus" pediu um trabalho com tema livre utilizando html, java, css e responsivo, eu decidi fazer com tema
do jogo "Clash royale", descrevendo meu "deck" de cartas do jogo e quais as estrátegias que eu utilizo com o "deck", utilizei IA para me ajudar, poisfoi permitido, no primeiro dia fiz o codígo bruto,
no segundo o "CSS", e no terceiro fiz ser responsivo.
